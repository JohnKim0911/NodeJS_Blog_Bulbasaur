## How to run

1. Download the code.
2. Open the project with IDE, run on its terminal:

      npm i
   
4. After downloading all the dependencies (number3), run this as well:

      nodemon index.js
   
5. Go to <a src="https://localhost:3000/" target="_blank">localhost:3000/</a>
